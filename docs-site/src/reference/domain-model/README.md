# Domain Model Statements

Statements for creating, altering, and dropping domain model elements: entities, enumerations, associations, and constants.

## Entity Statements

| Statement | Description |
|-----------|-------------|
| [CREATE ENTITY](create-entity.md) | Create a new entity in a module's domain model |
| [ALTER ENTITY](alter-entity.md) | Add, drop, modify, or rename attributes and indexes on an existing entity |
| [DROP ENTITY](drop-entity.md) | Remove an entity from the domain model |

## Enumeration Statements

| Statement | Description |
|-----------|-------------|
| [CREATE ENUMERATION](create-enumeration.md) | Create a new enumeration type with named values |
| [DROP ENUMERATION](drop-enumeration.md) | Remove an enumeration from the project |

## Association Statements

| Statement | Description |
|-----------|-------------|
| [CREATE ASSOCIATION](create-association.md) | Create a relationship between two entities |
| [DROP ASSOCIATION](drop-association.md) | Remove an association from the domain model |

## Constant Statements

| Statement | Description |
|-----------|-------------|
| [CREATE CONSTANT](create-constant.md) | Create a named constant value in a module |
